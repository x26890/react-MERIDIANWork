
# VTuber排班管理系統 #
使用者可以透過此系統輕鬆切換團隊身分、新增直播行程，並透過日曆介面即時掌握成員動態。

1.多團隊管理模式
    支援切換不同的團隊（如 Meridian 或 Spring Fish），系統會根據選擇自動過濾成員與行程。

2.即時同步日曆
    整合 react-calendar，並透過 CSS 優化格子顯示 。
    垂直捲動頭像，當單日行程過多時，日曆格子內會自動出現捲動條，顯示所有參與成員的頭像 。
    點擊日期可快速過濾當日行程列表 。

3.資料即時性
    Firebase Firestore：所有排班異動皆會即時同步至雲端資料庫。
    YouTube Data API v3：自動抓取成員最新的頻道頭像。

4.智慧排班表單
    內建重複行程檢查，防止同成員在同一時段重複排班。
    優化時間選擇器，支援 15 分鐘為單位的快速調整與 AM/PM 切換。

=================================================================================  

# 關鍵技術 #
前端框架: React(Vite)  

資料庫: Firebase Firestore 

外部 API: YouTube Data API v3  

樣式與 UI:Bootstrap 5 & Bootstrap Icons  

視覺化日曆:react-calendar  

Custom CSS (支援響應式手機版佈局)
